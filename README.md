# Recipe.com

Recipe.com is a simple recipe website created using HTML, CSS and JavaScript.  
It offers a clean interface to browse and view recipes, styled layouts and images for user friendly experience.

## Features
- View recipes by pictures
- Mobile friendly
- Quick to install and operate on the local machine
